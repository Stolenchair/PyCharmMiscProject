## Quick Start

### Installation
```bash
# Install dependencies
pip install pandas openpyxl

# Run application
python main.py
```

### Toggle Theme
1. Click menu: **View** ‚Üí **Dark Theme** (or **Light Theme**)
2. Theme is saved automatically
3. Restart for full effect

### Settings File
Theme saved to `prg_settings.json`:
```json
{
  "ui_preferences": {
    "theme": "dark"
  }
}
```

```

  ‚îÇ

  ‚îÇ
  ‚îú‚îÄ> StyleManager(theme='light' or 'dark')
  ‚îÇ     ‚îú‚îÄ> Light theme colors
  ‚îÇ     ‚îú‚îÄ> Dark theme colors
  ‚îÇ     ‚îî‚îÄ> Button styling methods

  ‚îî‚îÄ> PRGPipelineManager
```
